# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-08

### Added

- Initial project setup with Next.js 15, TypeScript, and TailwindCSS
- Agent-based architecture with standalone feature modules
- Catppuccin Mocha themed UI with glassmorphism effects
- HTTP Client agent with full request/response handling
- Collections agent with sidebar, favorites, history, and environments
- WebSocket agent with connection management and message logging
- Load Testing agent with simulated metrics and charts
- GraphQL agent with query/mutation/subscription support
- API Docs agent with markdown generation and preview
- Diff Viewer agent for response comparison
- Request Chain agent for chaining requests with variable extraction
- Security Scanner agent with JWT decoder
- Auto-Retry agent with circuit breaker logic
- Data Generator agent with template-based generation
- Collaboration agent with sharing and audit logging
- Profiler agent with performance metrics waterfall
- Mock Server agent for endpoint simulation
- Command Palette with keyboard shortcuts
- Request/Response interceptor with breakpoint debugging
- Toast notification system
- Quality-of-life tooling: Biome.js, Lefthook, Renovate, Knip
- Vitest test suite
